# Sales_Dashboard
This repository contains a comprehensive sales dashboard project designed to provide insights into sales performance and trends. The dashboard is built using power bi, allowing users to visualize and analyze sales data in an interactive and intuitive manner.

Features:
- Interactive visualizations showcasing sales data by region, product, and time period.
- Key performance indicators (KPIs) highlighting revenue, growth rates, and top-selling products.
- Drill-down capabilities for deeper insights into specific segments.
